# 反转链表

* **题目描述**

  输入一个链表，反转链表后，输出新链表的表头。

* **Code**

```java
/*
public class ListNode {
    int val;
    ListNode next = null;

    ListNode(int val) {
        this.val = val;
    }
}*/
public class Solution {
    public ListNode ReverseList(ListNode head) {
        if(head==null){return null;} 
         ListNode next = null;
         ListNode pre = null;
          while(head!=null){
              next = head.next;
              head.next = pre;
              pre = head;
              head = next;
          }
        return pre;
        }
    }

```
